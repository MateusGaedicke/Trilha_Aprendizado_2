Hiperlinks

Referência dentro de um documento em hipertexto para outras partes do mesmo documento
ou para outros documentos.

Tipos de Hiperlinks:

  • Hiperlink Interno

Link dentro de um documento(site) que faz referência a alguma parte do mesmo documento.
Sintaxe:
< a href="#texto" > texto linkado </a>
< tag id="texto"> Conteúdo para onde o link vai levar </tag >

  • Hiperlink Externo

Link de um documento(site) que faz referência a outro documento(site).
Sintaxe:
< a href="pasta/arquivo.formato_do_arquivo" > Texto Linkado < /a >
