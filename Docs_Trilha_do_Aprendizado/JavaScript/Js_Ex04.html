Exercício 01

A partir do número entrado pelo usuário determine se o número é par ou ímpar

Exercício 02

A partir de um número digitado pelo usuário determine se ele é divisível por 7 ou 3

Exercício 03

Faça um código que solicite ao usuário o nome e a idade. Apresente a mensagem "Bem vindo nome! Você tem xx anos". Caso ele seja maior de 18 anos o programa deve solicitar o CPF e exibir o número do CPF. Caso contrário deve exibir somente uma mensagem dizendo que é menor de idade.
