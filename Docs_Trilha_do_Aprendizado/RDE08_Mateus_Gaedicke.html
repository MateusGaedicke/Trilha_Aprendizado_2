Tags

Table

A tag < table > define uma tabela HTML. Dentro da tag < table > podemos ter um ou mais elementos
< tr >, < th > e < td >.

O elemento < tr > define uma linha de tabela, o elemento < th > define um cabeçalho de tabela e o
elemento < td > define uma célula de tabela.

Sintaxe:
< table >
  < tr >
    < th > Month < /th >
    < th > Savings < /th >
  &/lt tr >
  < tr >
    < td > January < /td >
    < td > $100 < /td >
 < /tr >
< /table >

Exemplo:

Month Savings
January $100

Del

A < del > tag define o texto que foi excluído de um documento. Os navegadores geralmente
riscam uma linha através do texto excluído.

Sintaxe:
My favorite color is < del > blue < /del > < ins > red < /ins >

Exemplo:
My favorite color is blue red

Ins

A tag < ins >define um texto que foi inserido em um documento. Os navegadores geralmente
sublinham o texto inserido.

Sintaxe:
My favorite color is < del > blue < /del > < ins > red < /ins >

Exemplo:
My favorite color is blue red

Sub

A tag < sub > define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha
normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para
fórmulas químicas, como H 2 O.

Exemplo:
This text contains subscript text.

Sup

A tag < sup > define o texto sobrescrito. O texto sobrescrito aparece meio caractere acima da
linha normal e, às vezes, é renderizado em uma fonte menor. O texto sobrescrito pode ser usado
para notas de rodapé, como WWW[1].

Exemplo:
This text contains superscript text.

Blockquote

A < blockquote > tag especifica uma seção que é citada de outra fonte. Os navegadores geralmente
recuam elementos < blockquote >.

Exemplo:

For 50 years, WWF has been protecting the future of nature. The world's leading conservation
organization, WWF works in 100 countries and is supported by 1.2 million members in the
United States and close to 5 million globally.

Q

A tag < q > define uma citação curta. Os navegadores normalmente inserem aspas ao redor da citação.

Exemplo:
WWF's goal is to: Build a future where people live in harmony with nature. We hope they succeed.

Abbr

A tag < abbr > define uma abreviação ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP",
"ATM". O atributo global < title > mostra a descrição da abreviatura/acrônimo ao passar o mouse sobre
o elemento.

Exemplo:
The WHO was founded in 1948.

Address

A tag < address > define as informações de contato do autor/proprietário de um documento ou artigo.
As informações de contato podem ser um endereço de e-mail, URL, endereço físico, número de telefone,
identificador de mídia social, etc. O texto no elemento < address > geralmente é renderizado em itálico
e os navegadores sempre adicionam uma quebra de linha antes e depois do elemento < address >.

Exemplo:

Written by Jon Doe.
Visit us at:
Example.com
Box 564, Disneyland
USA

Cite

A tag < cite > define o título de um trabalho criativo (por exemplo, um livro, um poema, uma música,
um filme, uma pintura, uma escultura, etc.). O texto no elemento < cite > geralmente é renderizado
em itálico.

Exemplo:
The Scream by Edward Munch. Painted in 1893.

Bdo

A tag < bdo > é usada para substituir a direção do texto atual.

Exemplo:
Texto original: This text will go right-to-left.
Texto com a tag < bdo >: This text will go right-to-left.

Code

A tag < code >é usada para definir um pedaço de código de computador. O conteúdo interno é exibido
na fonte monoespaçada padrão do navegador.

Exemplo:
The HTML button tag defines a clickable button.
The CSS background-color property defines the background color of an element.

Pre

A < pre > tag define o texto pré-formatado. O texto em um < pre > elemento é exibido em uma fonte
de largura fixa e o texto preserva espaços e quebras de linha. O texto será exibido exatamente como
escrito no código-fonte HTML.

Exemplo:

            Text in a pre element
            is displayed in a fixed-width
            font, and it preserves
            both      spaces and
            line breaks

Kbd

A tag < kbd > é usada para definir a entrada do teclado. O conteúdo interno é exibido na fonte
monoespaçada padrão do navegador.

Exemplo:
Press Ctrl + C to copy text (Windows).
